1.O que é HTML?

HTML, ou Linguagem de Marcação de Hipertexto, é a linguagem fundamental usada para criar e estruturar páginas web.
Basicamente, ela define a estrutura do conteúdo, como texto, imagens e outros elementos, dizendo ao navegador
como exibir a página. É importante ressaltar que o HTML não é uma linguagem de programação, mas sim uma linguagem
de marcação usada para organizar o conteúdo.

Objetivos Principais:

  1. Estruturar conteúdo (textos, imagens, vídeos).
  2. Definir semântica (títulos, parágrafos, listas, etc.).
  3. Criar ligações entre páginas (hyperlinks).
  4. Permitir acessibilidade (leitura por navegadores e ferramentas assistivas).
